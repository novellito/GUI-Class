Files:
1) CalculatorMain.java - Main class that initializes the application
2) CalculatorFrame.java - Class that builds the various components.
3) ButtonSetup.java - Creates the buttons along with its key listeners.
4) ClickListener.java - The Class that conains the main logic calculator logic of the application.
5) ButtonKeyListener.java - Sets up keylistener for keyboard input.
6) MenuSetup.java - Constructs the Menu of the app.
7) PanelSetup.java - Configures the panels of the app.

Functionality:
Standard Calculator functionality (+,-,*,/,%)
The user can click the buttons or press the appropriate keys
on their keyboard. 
Backspace will clear the last character and Enter will submit entry.
